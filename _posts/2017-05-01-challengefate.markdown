# 距离拼天赋还很遥远 #
> 有时候我会想，如果我有...样的天赋就好了。
 
期望自己拥有天赋，期望上苍眷顾自己，这是一种可恶的不劳而获的思想。
世人太过于关注天赋产生了唯天赋论，难道没有天赋就做不成事情了吗？如果是这样，
那么人的命运岂不是并非掌握在自己手中的吗？一个生命的旅程没有开始就要宣布结束了吗？

如果是这样的，做一个挑战苍天、挑战命运的人又有何不可？按唯天赋论者所说，命运中注定要不服于命运的吧。
